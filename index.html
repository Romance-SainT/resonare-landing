왜 더 잘 맞을까요?

소리가 빨리, 안 끊기게 도착해요

다른 프로그램은 소리가 멀리 돌아가서 늦게 도착하거나, 연결이 한 번 막히면 아예 안 될 때가 있어요. Resonare는 한국 안에서 가장 빠른 길로 잇고, 막혀도 다른 길로 자동으로 이어줘요.

😵 다른 합주 앱들

  • ✕연결을 도와주는 서버가 멀리(일본) 있어서, 소리가 늦게 도착할 때가 있어요
  • ✕연결이 한 번 막히면 대신 이어줄 길이 없어서, 어떤 인터넷에선 아예 연결이 안 돼요
  • ✕한국 인터넷 환경에 잘 안 맞을 때가 있어요
  • ✕방송 중에 초대 주소·비밀번호가 실수로 보일 수 있어요

✨ Resonare

  • ✓연결도, 막혔을 때 우회도 모두 한국 안에서 — 그래서 빠르고 잘 안 끊겨요
  • ✓서로 가장 빠른 길로 직접 연결 — 소리가 거의 바로 도착해요
  • ✓컴퓨터 소리 장치를 직접 써서 지연(밀림)을 최소화했어요
  • ✓초대 주소·비밀번호가 기본으로 가려져 있어 방송 사고를 막아요
기능

합주에 필요한 건 다 있어요

거의 실시간

소리가 눈 깜짝할 사이에 전달돼서, 멀리 있어도 바로 옆에서 하는 것처럼 박자를 맞출 수 있어요.

최대 10명 합주

서버를 거치지 않고 서로 직접 연결해서, 한 방에 최대 10명까지 같이 노래하고 연주해요.

끊겨도 자동 우회

직접 연결이 안 되는 인터넷 환경에서도, 한국 서버가 자동으로 이어줘서 잘 안 끊겨요.

방 안 채팅

합주하면서 같은 방 사람들과 메시지로 대화할 수 있어요. 곡 정하기, 박자 맞추기 편해요.

4개 언어

한국어 · English · 日本語 · 中文. 버튼 하나로 화면 언어를 바로 바꿀 수 있어요.

방송 사고 방지

초대 주소와 비밀번호가 기본으로 가려져 있어서, 방송 중에 실수로 노출되지 않아요.

소리 크기 한눈에

누가 소리를 내는지 막대로 보이고, 사람마다 볼륨을 따로 조절할 수 있어요.

쓰는 법

세 단계면 합주 시작

방 만들기

이름과 악기를 고르고 방을 만들어요. 원하면 비밀번호도 걸 수 있어요.

친구 초대

초대 주소를 친구에게 보내요. 방송 중이어도 안 보이게 복사돼서 안전해요.

같이 연주

친구가 들어오면 바로 연결돼요. 소리 막대를 보면서 같이 박자 맞춰 연주해요.
